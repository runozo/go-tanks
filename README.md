# go-tanks

A game with tanks<!-- >, try it on https://runozo.github.io/go-tanks/ -->

## Controls
|Key|Function
|------------|----------|
|Arrow keys|Move tank|
|a|Rotate barrel counterclockwise|
|d|Rotate barrel clockwise|
|Spacebar|Fire|
|T|New random tank|
|P|Redraw random playfield
